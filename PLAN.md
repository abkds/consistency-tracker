# WEEKDAY (Mon–Fri)

### Morning (fixed, no ML)

* **6:30 – 7:20** → Wake up, get ready
* **7:20 – 8:30** → Gym
* **8:30 – 9:30** → Commute / breakfast / get to office

---

### Workday (anchor)

* **9:30 – 5:30** → **Office ML work (6 hours)**

---

### Evening (home by ~7:00)

* **5:30 – 7:00** → Commute + dinner + decompress

---

### Study block (simple, repeatable)

* **7:30 – 9:00** → **90 minutes total**

**Weekly split**:

| Day     | 7:30–9:00                 |
| ------- | ------------------------- |
| **Mon** | HOML3                     |
| **Tue** | HOML3                     |
| **Wed** | Karpathy                  |
| **Thu** | Papers                    |
| **Fri** | Rest / catch-up / nothing |

Inside that 90 min:

* 10 min recall
* 70 min focused
* 10 min notes
